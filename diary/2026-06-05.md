- `megatron_runtime.py` 已有完整的 attention hook (RoPE扩展、KV扩展、因果掩码)
- `layer_id` 在 hook 中已通过 `attention_module.layer_number` 获取, 可用于层类型路由
- flash-attn 编译可能需要 30+ 分钟 (RTX 4090, CUDA 12.8)
- GitHub 网络间歇性不可用, 需要在可用时及时 push
- **GQA QKV interleaving**: Megatron 的 linear_qkv 输出不是简单的 [Q|K|V] 拼接, 而是 [q_per_group|k|v] per query group 的交错格式。必须使用 `get_query_key_value_tensors()` 而不是手动 `torch.split`
