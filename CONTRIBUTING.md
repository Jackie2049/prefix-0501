# Contributing

感谢你关注 PrefixSharing！本项目目前为论文附件的快速开源版本，暂未开放完整的开发者贡献流程。

## 报告问题

欢迎通过 GitHub Issues 提交 bug 报告或功能建议。

## 贡献代码

如你希望贡献代码，请先开一个 Issue 讨论你的想法。当前阶段我们优先关注：

- Bug fix
- 精度对齐验证
- 新后端支持（flash attention 等）

提交 PR 时请确保：

1. 测试通过（见 AGENTS.md 中的测试命令）
2. 保持 commit message 格式：`[type] 简要说明`
   - type: feat / fix / refactor / test / doc / chore
3. PR body 包含 `## 测试结果` 小节
